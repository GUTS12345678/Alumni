Login Debug Test

This page will test various login scenarios

Test Links:

  • Direct Login Page
  • Home Page
  • Admin Dashboard (should redirect to login)

Instructions:

  1. First, clear your browser cookies/session for localhost:8000
  2. Click the "Direct Login Page" link
  3. It should show the login form, not redirect to home
  4. If it redirects, there might be a middleware issue

Debugging Steps:

1. Open browser developer tools (F12)

2. Go to Network tab

3. Click login link and check for redirects

4. Check if there are any JavaScript errors in Console tab
